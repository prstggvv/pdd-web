## Установка проекта
Клонирование проекта SSH:
```sh
$ git clone 
```

Установка зависимостей:
```sh
$ npm i
```
___

## Запуск проекта
Запуск проекта:
```sh
npm run start
```

development сборка:
```sh
npm run build:dev
```

production сборка: 
```sh
npm run build
```
____

## Проверка линтинга
Для проверки TypeScript-файлов используется ESLint с конфигурацией airbnb-base.

Проверка файлов:
```sh
npm run lint
```

Исправить файлы:
```sh
npm run lint:fix
```

Для проверки CSS-файлов используется Stylelint.

Проверить css-файлы:
```sh
npm run lint:css
```

Исправить css-файлы:
```sh
npm run lint:css:fix
```
